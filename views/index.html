Timestamp Microservice

What this app does:

This web app provides a simple API that can convert dates into a Unix timestamp or a UTC date format. It can process two types of input:

  • Date String: You can provide a human-readable date (e.g., "2015-12-25") to get back a Unix timestamp (number of milliseconds since Jan 1, 1970) and the date in a standardized UTC format.
  • Unix Timestamp: You can also provide a Unix timestamp (e.g., 1451001600000), and it will return the date in both Unix timestamp and UTC formats.

To use the API, you can either manually add /api at the end of the URL with a date or timestamp (e.g., [project url]/api/2015-12-25) or simply click on the links below to try the examples.
